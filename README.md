# 🚀 Bootstrap 5 Advanced UI Project

## 📌 Project Overview

This project is a fully responsive multi-page web application developed using **Bootstrap 5**. It demonstrates modern UI design, component integration, and responsive layout techniques by combining various Bootstrap elements into a clean and professional website.

---

## 🎯 Features

* ✅ Responsive design (Mobile, Tablet, Desktop)
* ✅ Modern Hero Section with background image
* ✅ Navigation bar with multiple pages
* ✅ Portfolio section with real images
* ✅ Skills section with icons
* ✅ Contact form with validation
* ✅ Login page with basic authentication (frontend)
* ✅ Dashboard UI layout
* ✅ Dark mode toggle 🌙
* ✅ Smooth animations using AOS library
* ✅ Clean and structured UI/UX design

---

## 🛠️ Technologies Used

* **HTML5**
* **CSS3**
* **Bootstrap 5**
* **JavaScript**
* **Bootstrap Icons**
* **AOS (Animate on Scroll Library)**
* **Git & GitHub**

## ⚙️ How the Project Was Built

1. Explored Bootstrap 5 documentation and examples.
2. Selected UI components such as Navbar, Cards, Forms, and Grid.
3. Designed multiple pages (Home, About, Contact).
4. Added advanced features like:

   * Dark mode toggle
   * Animations (AOS)
   * Login system (frontend)
   * Dashboard UI
5. Applied custom CSS for better styling and layout consistency.
6. Tested responsiveness across different screen sizes.
